# Python Workflow

This folder will contain the Python scripts used for chart creation, regression modelling, backtesting and diagnostics.

The Python scripts will be added after the final report outputs are confirmed.

Planned scripts:

1. `01_part1_charts_and_forecast.py`
   - creates Part 1 charts
   - calculates forecast spread values
   - prepares visual outputs for the baseline projection

2. `02_part2_regression_models.py`
   - runs the regression models used in Part 2
   - includes raw linear time trend regression, semi log regression, lagged semi log regression and hedonic regression

3. `03_backtesting_and_diagnostics.py`
   - runs backtesting checks
   - calculates RMSE and MAE
   - checks residuals, Durbin Watson and VIF diagnostics

The final Python files will be added after Part 1 and Part 2 are fully reviewed.
